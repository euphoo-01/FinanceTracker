# Finance Tracker

_Исходные файлы проекта находятся в папке src/_

### Краткое описание:

Пет-проект, финансовый помощник. Можно отслеживать собственные расходы и просматривать наглядно на инфографике.

### Как запустить?

На компьютере должен быть установлен NodeJS

1. Клонируйте репозиторий на свою машину
2. `npm ci`
3. `npm run dev`
4. Готово!

### Стек:

- TypeScript
- React
- Zustand
- Recharts

### Приёмы:

- Custom Hooks (useForm, useBalance, useTransaction, useNotification)
- Простая модульная архитектура
- CSS Modules
- Lazy loading
- Generic Types (для переиспользуемости useForm)
